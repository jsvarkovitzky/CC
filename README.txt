
Crescent City Tsunami Modeling

Before running code, do:
   python get_topo.py
to retrieve topo and dtopo files from geoclaw website.

template-tohoku-fujii  
   contains a template for a run using the Tohoku
   source model of Fujii et. al.

template-tohoku-ucsb3
   identical but uses UCSB Version 3 model.

template-tohoku-restart
   has a version where a restart is attempted, but not fully working yet.
